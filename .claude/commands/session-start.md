# 会话启动检查清单

开发会话开始时，请按顺序进行以下操作：

## 1. 阅读开发计划

请阅读 `documentation/PLAN.md` 文件，确认本次会话的目标和优先级功能。

## 2. 查看开发进度

请查看 `documentation/PROGRESS.md` 文件，了解架构决策背景和最近会话的技术选择。

## 3. 检查文档维护工作

请检查：
- [ ] 是否需要归档旧会话（PROGRESS.md 超过 4 个会话）
- [ ] 是否需要移动已完成功能到 `archive/COMPLETED_FEATURES.md`
- [ ] 文档维护规则是否遵守（查看 `documentation/memos/DOCUMENTATION_MAINTENANCE.md`）

---

**准备就绪后，告诉我本次会话的目标。**
